# Anime Feature Extraction Project

## Overview
This project focuses on feature extraction from the anime.csv dataset using Python, NumPy, and Pandas. The objective is to extract meaningful numerical and categorical features that can be used for further analysis or modeling.

## Dataset
- File: anime.csv
- Contains information related to anime titles, genres, ratings, and other attributes.

## Key Work
- Loading and inspecting the dataset
- Handling missing and inconsistent values
- Extracting relevant numerical features
- Extracting and transforming categorical features
- Data manipulation using NumPy and Pandas

## Tech Stack
- Python
- NumPy
- Pandas
